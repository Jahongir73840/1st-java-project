## Description
This project is a simple Spring Boot application created as part of Task 1.
It demonstrates basic usage of Spring Boot, controllers, Thymeleaf templates,
and static resources.

## Technologies
- Java
- Spring Boot
- Spring Web
- Thymeleaf
- Maven

## How to run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run `SpringProjectApplication`.
4. Open:
   - http://localhost:8080/
   - http://localhost:8080/greeting
